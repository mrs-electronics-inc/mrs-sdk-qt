This guide walks you through the steps of installing all of the SDK components from source, including the Qt libraries and tooling.

<Aside type="tip">
The preferred installation method is to follow [this installation guide](/guides/installation), which uses prebuilt releases.

Building from source is rarely a necessary step. This guide is provided for a) power users and b) people having trouble with the normal installation process.
</Aside>

## Install Dependencies

You will need to have Qt5/Qt6 installed before you can compile.

## Install the SDK

Here are the steps for installing from source:

<Steps>

1. Clone the SDK repository:

    ```bash
    git clone https://github.com/mrs-electronics-inc/mrs-sdk-qt.git
    cd mrs-sdk-qt
    ```

2. Install the `just` command runner ([read the docs](https://just.systems/man/en/quick-start.html)):

    <Tabs>
        <TabItem label="Debian">
            ```bash
            apt install just
            ```
        </TabItem>
        <TabItem label="NPM">
            ```bash
            npm install -g rust-just
            ```
        </TabItem>
        <TabItem label="Pipx">
            ```bash
            pipx install rust-just
            ```
        </TabItem>
    </Tabs>

3. Compile and install the SDK tooling:

    ```bash
    just tools
    ```

4. Compile and install the SDK Qt libraries:

    ```bash
    just libs
    ```

</Steps>
