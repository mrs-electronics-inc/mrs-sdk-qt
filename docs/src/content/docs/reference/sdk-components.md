Note that all directory paths in this reference are described relative to the top level of the installed version.

## Qt Libraries

The `mrs-sdk-qt` Qt/C++ library is the centerpiece of the SDK.

### Headers

Located in `include`

### Build System

#### CMake

Located in `lib/cmake`

#### QMake

Located in `lib/qmake`

### Compiled Libraries

Located in `lib/qt5` and `lib/qt6`

## Demos

Located in `demos`

## Tooling

Located in `bin`

## Documentation

Located in `docs`
