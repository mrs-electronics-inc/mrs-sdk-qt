# MRS SDK Qt - Virtual Machine

We provide a pre-built VM image optimized for developing applications with `mrs-sdk-qt`.

## Quick Links

- **Full Setup Guide:** See [Virtual Machine Setup](https://qt.mrs-electronics.dev/guides/set-up-vm/) for complete instructions on obtaining, importing, and using the VM
- **Download:** [MRS SDK Qt VM Images on OneDrive](https://mrselectronics-my.sharepoint.com/:f:/g/personal/addison_emig_mrs-electronics_com/EmT5AxglIxBJnDWp7DWMYTgBqBoZhU_oodHmsWTj_M0EEQ?e=SBnOGw)
- **Build Instructions:** See [BUILD.md](./BUILD.md) to build the VM image locally

## Overview

Ubuntu 24.04 LTS Desktop (minimal) with:

- Qt Creator IDE
- Qt 5.15 and Qt 6.8 development kits
- Build tools (`gcc`, `g++`, `make`, `cmake`, etc)
- MRS Qt SDK
  - WIP, coming soon

## Getting Started

For comprehensive setup instructions, see the [Virtual Machine Setup guide](https://qt.mrs-electronics.dev/guides/set-up-vm/), which covers:

- Obtaining the VM image (download or build)
- Importing into your hypervisor
- First boot and provisioning
- Using Qt Creator
- Customization and troubleshooting
